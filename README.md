# dbpal

Your friendly database companion - A powerful tool that uses LLM to help with database operations.

## Installation

### From PyPI
```bash
pip install dbpal
```

### From Source
```bash
git clone https://github.com/pie111/db-pal.git
cd db-pal
pip install -e .
```

## Usage

### Basic Commands
```bash
# Start dbpal
dbpal
```
